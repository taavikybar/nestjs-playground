## Description

[Nest](https://github.com/nestjs/nest) + [MongoDB](https://www.mongodb.com/) + [Nats](https://nats.io/) playground 

## Installation

```bash
$ npm i
```

## Running the app

```bash
# start mongodb and nats containers
docker-compose up -d

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
